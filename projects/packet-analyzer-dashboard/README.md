Packet Analyzer Dashboard
Overview
Packet Analyzer Dashboard is a beginner-friendly cybersecurity and networking project built with Python, Flask, HTML, and CSS.
The goal of this project is to help analyze basic network traffic information and present it in a simple dashboard format.
Why I Built This
As a cybersecurity student studying CCNA, I wanted to build a project that connects networking theory with real security analysis.
Packet analysis is useful for:
Network troubleshooting
SOC analysis
Incident response
Detecting suspicious traffic
Understanding protocols
Features
Displays sample packet data
Shows source IP and destination IP
Shows protocol type
Shows destination port
Labels traffic as normal or suspicious
Provides a simple dashboard interface
Technologies Used
Python
Flask
HTML
CSS
What I Learned
Through this project, I learned how packet information can be organized and analyzed. I also practiced connecting networking concepts such as IP addresses, protocols, and ports with cybersecurity monitoring.
