# CCNA 1 - Network Fundamentals

## What is a Network?

A network is a collection of devices connected together to share information and resources.

Examples include:

* Home Wi-Fi networks
* University networks
* Company networks
* The Internet

---

## Why Do We Need Networks?

Networks allow devices to:

* Communicate with each other
* Share files
* Access the Internet
* Share printers and resources
* Support applications and services

Without networks, modern communication would not be possible.

---

## Types of Networks

### LAN (Local Area Network)

A network covering a small geographic area.

Examples:

* Home network
* School lab
* Office floor

Characteristics:

* High speed
* Privately managed
* Limited coverage

---

### WAN (Wide Area Network)

A network covering large geographic areas.

Examples:

* The Internet
* Connections between company branches

Characteristics:

* Large coverage area
* Uses service providers
* Connects multiple LANs

---

### WLAN (Wireless Local Area Network)

A LAN that uses wireless communication.

Examples:

* Wi-Fi networks
* Mobile hotspots

---

## Network Devices

### Router

A router connects different networks together.

Functions:

* Routes traffic between networks
* Connects a LAN to the Internet
* Makes forwarding decisions using IP addresses

---

### Switch

A switch connects devices within the same network.

Functions:

* Learns MAC addresses
* Forwards frames efficiently
* Operates primarily at Layer 2

---

### Access Point (AP)

Provides wireless connectivity.

Functions:

* Allows wireless devices to join the network
* Extends network access through Wi-Fi

---

### Firewall

A security device that controls network traffic.

Functions:

* Filters packets
* Blocks unauthorized access
* Protects internal networks

---

## End Devices

End devices are devices used by users.

Examples:

* PCs
* Laptops
* Smartphones
* Printers
* Servers

These devices generate and receive data.

---

## Network Media

Network media carries data between devices.

### Copper Cables

Examples:

* Ethernet cables

Advantages:

* Low cost
* Easy installation

Disadvantages:

* Limited distance

---

### Fiber Optic Cables

Advantages:

* Very high speed
* Long distances
* Resistant to interference

Disadvantages:

* More expensive

---

### Wireless Media

Examples:

* Wi-Fi
* Cellular networks

Advantages:

* Mobility

Disadvantages:

* More interference
* Lower security than wired networks

---

## Bandwidth

Bandwidth is the maximum amount of data that can be transmitted over a network connection in a given period.

Measured in:

* Kbps
* Mbps
* Gbps

Example:

A 1 Gbps link can theoretically transfer up to 1 billion bits per second.

---

## Latency

Latency is the delay between sending and receiving data.

Lower latency means faster communication.

Examples:

* Online gaming
* Video conferencing

---

## Client-Server Model

In this model:

### Server

Provides services and resources.

Examples:

* Web server
* Email server
* DNS server

### Client

Requests services from the server.

Examples:

* Web browser
* Email application

---

## Peer-to-Peer (P2P)

Devices communicate directly without a dedicated server.

Advantages:

* Simple
* Low cost

Disadvantages:

* Harder to manage
* Less scalable

---

## CCNA Exam Tips

Remember:

* Routers connect networks.
* Switches connect devices.
* WANs connect multiple LANs.
* Bandwidth is capacity.
* Latency is delay.
* Access points provide wireless connectivity.

## Summary

Network fundamentals are the foundation of CCNA. Understanding network types, devices, media, bandwidth, and communication models is essential before studying IP addressing, switching, routing, and network security.
