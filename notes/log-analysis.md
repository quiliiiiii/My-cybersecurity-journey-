# Log Analysis Basics

## What is Log Analysis?

Log analysis is the process of reviewing system logs to identify:

- Failed logins
- Malware activity
- Unauthorized access
- Suspicious behavior

## Why It Matters

Security analysts spend a large amount of time reviewing logs.

Logs provide evidence of attacks and system activity.

## What I Built

I created a Python log analyzer that:

- Reads authentication logs
- Detects failed login attempts
- Counts attacks per IP address
- Flags suspicious sources
