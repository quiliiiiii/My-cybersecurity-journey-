# Networking Fundamentals

## What is a Network?

A network is a group of devices connected together to exchange data.

Examples:

* Home Wi-Fi
* University network
* The Internet

## Important Concepts

### IP Address

An IP address identifies a device on a network.

Example:
192.168.1.10

### MAC Address

A unique hardware address assigned to a network interface.

### DNS

DNS translates domain names into IP addresses.

Example:

google.com → 142.250.x.x

### TCP vs UDP

TCP:

* Reliable
* Connection-oriented
* Used by HTTP, HTTPS, SSH

UDP:

* Faster
* Connectionless
* Used by DNS, streaming, gaming

## Common Ports

| Port | Service |
| ---- | ------- |
| 22   | SSH     |
| 53   | DNS     |
| 80   | HTTP    |
| 443  | HTTPS   |

## Useful Commands

```bash
ping google.com
ipconfig

```
