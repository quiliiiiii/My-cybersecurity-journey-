# Ports and Port Scanning

## What is a Port?

A port is a logical communication endpoint on a computer.

Common ports:

- 22 SSH
- 80 HTTP
- 443 HTTPS
- 53 DNS

## What is Port Scanning?

Port scanning is the process of checking which ports are open on a system.

Security professionals use it to:

- Discover services
- Identify attack surfaces
- Perform security assessments
